HTML 教程

HTML超文本标记语言(HyperText Markup Language)是一种用于创建网页的标准标记语言.

您可以使用HTML来建立自己的WEB站点,HTML运行在浏览器上,由浏览器来解析.

HTML标签 展示相关

  • 标题 h1 - h6
  • 段落 p
  • 换行 br
  • 文本格式化
    • b 定义粗体文字
    • em 定义着重文字
    • i 定义斜体文字
    • small 定义小号字
    • strong 定义加重语气
    • sub 定义下标字
    • sup 定义上标字
    • ins 定义插入字
    • del 定义删除字
  • 计算机输出标签
    • pre 定义可换行文本
    • code 定义计算机代码
    • kbd 定义键盘码
    • samp 定义计算机样本
  • 引文,引用,及标签定义
    • abbr 定义缩写
    • address 定义地址
    • bdo 定义文字方向 定义文字方向
    • blockqubote 定义长引用
    • q 定义短引用
    • cite 定义引用,引证
    • dfn 定义一个项目
  • 链接 a
  • 图像 img
  • 水平线 hr
  • 注释 <!-- -->
  • head 元素包含了所有头部的标签元素
    • title 定义文档标题
    • base 描述了基本的链接地址,文档中的所有链接在此基础上拼接. test
    • link 定义了文档与外部资源之间的关系
    • style 定义了文档与外部样式文件之间的关系
    • meta 提供了元数据,元数据也不显示在页面上,但会被浏览器解析.
    • script 用于加载脚本文件
  • 表格 table 数据单元格可以包含文本,图片,列表,段落,水平线,表格等等.
  • 列表 无序ul 有序 ol 自定义 dl
  • 区块
    • 区块元素 块级元素在浏览器显示时通常会以新的行来开始
    • 内联元素 内联元素在显示时通常不会以新的行开始
    • 块级元素div 可用于组合其他HTML 元素的容器
    • 内联元素span 可用于文本的容器 当与css一同使用时,可用于设置样式属性.

HTML标签 输入相关

标签的属性

  • class 为html元素定义一个或多个类名(类名从样式文件引入)
  • id 定义元素的唯一id
  • style 规定元素的行内样式 (inline style)
  • title 描述了元素的额外信息(作为工具条使用)

css 样式

  • 背景颜色 background-color
  • 字体 font-family
  • 字体大小 font-size
